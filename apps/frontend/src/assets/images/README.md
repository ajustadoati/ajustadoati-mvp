# Imágenes de la Aplicación

## Logo de AjustadoATi

Para agregar el logo oficial de AjustadoATi:

1. Guarda el logo del brand en formato PNG
2. Nómbralo como `logo.png`
3. Colócalo en este directorio: `src/assets/images/`
4. El tamaño recomendado es **512x512 píxeles** (se escalará automáticamente)

## Formatos Soportados

- PNG (recomendado para transparencia)
- JPG/JPEG
- SVG (ideal para logos)
- WebP

## Ubicaciones donde se usa el logo

- **Pantalla de Login**: Logo principal en la parte superior
- **Splash Screen**: (si se configura)
- **Favicon**: Logo en el navegador
